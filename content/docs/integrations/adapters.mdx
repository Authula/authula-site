# Overview

GoBetterAuth provides adapters for some popular Go HTTP routers and frameworks, allowing you to easily integrate authentication into your applications. These adapters are designed to work seamlessly with the GoBetterAuth handler, making it easy to mount the authentication routes on your preferred router or framework. Some have adapters and others can be used directly with the handler without needing an adapter.

> **Note:** When using adapters, any custom routes you create outside the GoBetterAuth handler won't have access to the GoBetterAuth hooks system. To make use of hooks on your custom routes, use the method `auth.RegisterCustomRoute()`.

## Standard Library (net/http)

The following example shows how to use the GoBetterAuth handler with the standard library's `net/http` package. This doesn't require an adapter and can be used directly with the handler provided by GoBetterAuth.

```go
package main

import (
	"log"
	"net/http"

	gobetterauth "github.com/GoBetterAuth/go-better-auth/v2"
	gobetterauthconfig "github.com/GoBetterAuth/go-better-auth/v2/config"
	gobetterauthmodels "github.com/GoBetterAuth/go-better-auth/v2/models"
)

func main() {
	config := gobetterauthconfig.NewConfig(/*...*/)
	auth := gobetterauth.New(&gobetterauth.AuthConfig{
		Config:  config,
		Plugins: []gobetterauth.Plugin{},
	})

	// Mount handler
	http.Handle("/api/auth/", auth.Handler())

	// Start the server and mount the handler
	log.Fatal(http.ListenAndServe(":8080", nil))
}
```

## Chi

GoBetterAuth already uses Chi under the hood. The following example shows how to add more routes to GoBetterAuth. This doesn't require an adapter and can be used directly with the handler provided by GoBetterAuth.

```go
package main

import (
	"log"
	"net/http"

	gobetterauth "github.com/GoBetterAuth/go-better-auth/v2"
	gobetterauthconfig "github.com/GoBetterAuth/go-better-auth/v2/config"
	gobetterauthmodels "github.com/GoBetterAuth/go-better-auth/v2/models"
)

func main() {
	config := gobetterauthconfig.NewConfig(/*...*/)
	auth := gobetterauth.New(&gobetterauth.AuthConfig{
		Config:  config,
		Plugins: []gobetterauth.Plugin{},
	})

	// Add a custom route. This will be added outside of auth routes such as /api/auth
	// allowing you to define your own custom routes.
	auth.RegisterCustomRoute(gobetterauthmodels.Route{
		Method:   "GET",
		Path:     "/api/health",
		Metadata: map[string]any{},
		Handler: http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
			reqCtx, _ := gobetterauthmodels.GetRequestContext(r.Context())
			reqCtx.SetJSONResponse(http.StatusOK, map[string]any{
				"status": "ok",
			})
		}),
	})

	log.Fatal(http.ListenAndServe(":8080", auth.Handler()))
}
```

## Echo

The following example shows how to use Echo to mount the GoBetterAuth handler. This allows you to easily integrate GoBetterAuth into your Echo-based applications and doesn't require an adapter since the handler can be used directly.

```go
package main

import (
	"log"
	"net/http"

	"github.com/labstack/echo/v4"

	gobetterauth "github.com/GoBetterAuth/go-better-auth/v2"
	gobetterauthconfig "github.com/GoBetterAuth/go-better-auth/v2/config"
	gobetterauthmodels "github.com/GoBetterAuth/go-better-auth/v2/models"
)

func main() {
	config := gobetterauthconfig.NewConfig(/*...*/)
	auth := gobetterauth.New(&gobetterauth.AuthConfig{
		Config:  config,
		Plugins: []gobetterauth.Plugin{},
	})

	echoInstance := echo.New()

	// Mount auth handler
	echoInstance.Any("/api/auth/*", echo.WrapHandler(auth.Handler()))

	log.Fatal(echoInstance.Start(":8080"))
}
```

## Fiber

The following example shows how to use the official Fiber adapter to mount the GoBetterAuth handler on a Fiber application.

```go
package main

import (
	"log"

	"github.com/gofiber/fiber/v3"

	gobetterauth "github.com/GoBetterAuth/go-better-auth/v2"
	gobetterauthconfig "github.com/GoBetterAuth/go-better-auth/v2/config"
	gobetterauthmodels "github.com/GoBetterAuth/go-better-auth/v2/models"
	fiberadapter "github.com/GoBetterAuth/go-better-auth/v2/adapters/fiber"
)

func main() {
	config := gobetterauthconfig.NewConfig(/*...*/)
	auth := gobetterauth.New(&gobetterauth.AuthConfig{
		Config:  config,
		Plugins: []gobetterauth.Plugin{},
	})

	app := fiber.New()

	// Mount GoBetterAuth handler using the Fiber adapter
	app.Use("/api/auth", fiberadapter.New(fiberadapter.Config{
		Handler: auth.Handler(),
	}))

	// Add your custom routes here using app.Get, app.Post, etc.

	log.Fatal(app.Listen(":8080"))
}
```
