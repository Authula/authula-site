## What's New

- A big update to the Node.js SDK. The NPM library now integrates [Orval](https://orval.dev) to auto-generate all types and methods for the SDK based on Authula's OpenAPI specification whilst wrapping it to provide it with our custom Authula SDK.
- This is also a breaking change as the SDK has naturally changed due to this and there are changes that need to be applied to your codebase. Check out the NPM page for further details.

[NPM](https://www.npmjs.com/package/authula)
